Discriminación en el Sistema Penal Mexicano

Proyecto reproducible de análisis cuantitativo sobre posibles desigualdades en el acceso a la justicia y resultados del proceso penal en México.

El objetivo es evaluar cómo variables sociodemográficas y condiciones del debido proceso se asocian con el resultado judicial, utilizando modelos de regresión y análisis estadístico en R.
Objetivo

Analizar si características como edad, género, autoadscripción indígena, orientación sexual y condiciones procesales (defensor, presencia del juez, explicación de derechos) se relacionan con diferencias en el desenlace del proceso penal.

Metodología

Limpieza y normalización de bases de datos

Construcción de variables categóricas

Modelos de regresión (por sexo y modelo conjunto)

Análisis comparativo de coeficientes

Interpretación sustantiva con enfoque en derechos humanos

Estructura del proyecto

scripts/ : Limpieza, modelado y análisis estadístico

visuals/ : Resultados exportados

reports/ : Reportes de modelos

Reproducibilidad

El análisis está estructurado para ser replicable.
